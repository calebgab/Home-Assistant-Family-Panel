IP Allowlist
Only devices on this list can load the dashboard. Leave empty to allow all LAN devices. Localhost (127.0.0.1) is always allowed regardless.
Blocked IPs

Photo Frame
When idle, the dashboard becomes a fullscreen photo slideshow powered by Immich.
Immich URL
Base URL of your Immich server, e.g. http://192.168.1.10:2283
API Key
Immich → Account Settings → API Keys → New API Key
Required scopes: asset:read asset.view album:read user.read
Album
The album to show — e.g. your Favourites album
Idle timeout
Minutes of inactivity before slideshow starts (0 = disabled)
Seconds per photo
How long each photo is shown
Tap zones
Left ¼ = previous photo, right ¼ = next photo, middle ½ = exit. When off, tap anywhere exits.
